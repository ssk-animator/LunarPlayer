# Feature Priority

## P0 — Must Exist (MVP)

- [ ] Open Video — drag & drop, file dialog, command line
- [ ] Play — start playback from current position
- [ ] Pause — freeze on current frame
- [ ] Seek — jump to any position via slider or keyboard
- [ ] Volume — up, down, mute toggle
- [ ] Fullscreen — toggle fullscreen mode

---

## P1 — Important

- [ ] Hardware Decode — NVIDIA NVENC, Intel QSV, AMD AMF, VAAPI
- [ ] Frame Step — J/K/L or Left/Right arrow, one frame at a time
- [ ] Timeline Zoom — infinite zoom into timeline (like Blender VSE)
- [ ] Screenshots — export current frame to PNG/JPG

---

## P2 — Creator Features

- [ ] Histogram — luminance and RGB parade
- [ ] Waveform — luma waveform monitor
- [ ] LUT — load and apply 1D/3D LUTs
- [ ] Compare Mode — side-by-side, wipe, A/B toggle

---

## P3 — Advanced

- [ ] EXR Sequences — GPU-accelerated OpenEXR playback
- [ ] Version Stacks — auto-detect v001, v002, v003 sequences
- [ ] Difference Viewer — pixel-level diff between two frames
