# Lunar Player Vision

## Mission

Lunar Player is a modern GPU-accelerated media player designed for both consumers and creators.

## Core Principles

1. **Playback First** — Start playing immediately on open. No loading screens.
2. **Analyze Later** — Advanced analysis tools exist but never block playback.
3. **Frame Accuracy** — Every seek, step, and pause lands on an exact frame.
4. **GPU First** — All pixel work happens on the GPU. The CPU manages control flow only.
5. **Zero Clutter** — Default UI shows transport controls only. No toolbars, no panels.
6. **Massive File Ready** — Open 100GB+ files without pre-processing or indexing.
7. **Creator Tools Hidden Until Needed** — Press F3 to reveal creator mode. Everything else stays out of the way.

## Positioning

> Fast enough for everyday users. Accurate enough for artists.

## Product Vision

> Instant media review for everyone.
